Share Juice Plugin helps you create various social media buttons delivered from sites such as facebook, Twitter, Google Plus etc. You can use these buttons on a floating bar or before content or after content. These buttons can also be put in a specific order and 
different configuration used at three diffrent location. 
== Description ==
It is very simple to use this plugin. 
After installation, go to Share Juice->Social Buttons and start creating the buttons with your preferred configurations. At the end of each button there is link which take you to the individual site.
Choose location where each button will be shown. 
Use global options to define which share button group you want to be visible on main blog. Thats it
More documentation at <a href="http://www.webtecho.com/share-juice">Webtecho.com</a>

== Installation ==

 1.   Download the ZIP package from the wordpress plugin directoty
 2.   Log in to your WordPress admin  and upload the zip file using the built in plugin uploader.
 4.   Go to the “Plugins” page in your dashboard.
 5.   Activate the “Share Juice” plugin by clicking the “Activate” link.
 6.   Start creating the buttons and configure the options to use them. 

If you are using cache plugins, do not forget to emtpy the cache and especially js minify settings.
Also take backup of your database before you decide to install the plugin
